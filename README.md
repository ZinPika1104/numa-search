# ぬまサーチ 🔍

新潟県南魚沼市の観光スポット検索アプリ。定番からニッチまで367スポットを収録。

**公開ページ**: https://zinpika1104.github.io/numa-search/

## 機能

- キーワード・ジャンル検索(観光/自然/飲食/カフェ/温泉/宿泊/買物/心霊)
- スポット詳細(ニッチ情報・Googleマップリンク・周辺スポット提案)
- マイルート作成(近い順並べ替え・Googleマップでルート起動)
- おすすめモデルコース / 南魚沼の四季 / ぬまクイズ(全10問)
- ぬまマップ(ジャンル色分けピンの地図表示)

## 編集方法

`index.html` 内の `SPOTS` 配列を編集するだけでスポットを追加・修正できます。
詳しくは `CLAUDE.md`(Claude Code用の引き継ぎ資料)を参照。

## データについて

- 営業時間・料金は変動します。お出かけ前に公式情報をご確認ください
- 座標は近似値です(周辺提案・所要時間の目安計算用)
