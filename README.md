# ぬまサーチ 🏔

新潟県南魚沼市の観光スポット検索アプリ。定番からニッチ・穴場まで367スポットを収録。

**👉 アプリを開く: `index.html` をブラウザで開くだけ（インストール不要）**

## 機能

- ジャンル検索（観光・自然・飲食・カフェ・温泉・宿泊・買物・心霊）＋キーワード検索
- スポット詳細（ニッチ情報・Googleマップリンク・近くのスポット提案）
- 🚗 マイルート作成（現在地からのルートをGoogleマップで開ける）
- 🗺 おすすめモデルコース
- 🍃 南魚沼の四季ガイド
- 🧠 ぬまクイズ（ご当地クイズ全10問）

## 編集方法

すべて `index.html` 1ファイルで完結しています。

- **スポットの追加・修正**: `SPOTS` 配列を編集
- **クイズの追加**: `QUIZ` 配列を編集
- **四季の写真**: `SEASONS` 配列の `photo:""` に画像URLを入れる
- **モデルコース**: `COURSES` 配列を編集

編集したらブラウザで開いて動作確認 → コミット。

## データについて

- 座標は一部エリア近似値（地図リンクは名前検索なので正確な場所が開きます）
- 営業時間・料金は変わることがあります。お出かけ前に公式情報を確認してください
- スポット情報は南魚沼市公式観光サイト等を参考に独自に編集したものです
